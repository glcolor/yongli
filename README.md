# 永历纪年计算器 (Yongli Calendar Calculator)

一个用于永历纪年与其他纪年系统（公元、民国、伪清年号）相互换算的网页工具。

## 项目简介

永历纪年计算器是一个基于 Web 的交互式工具，专门用于进行永历纪年与其他常见纪年系统之间的换算。永历是南明时期使用的年号，始于公元1647年（永历元年）。本工具可以帮助用户快速进行不同纪年系统之间的转换。

## 局限性

本项目未考虑中国传统使用的农历纪年与公元纪年之间的差异，仅以“年”为单位进行粗略换算。

## 使用方法

直接打开 `index.html` 文件即可使用，无需安装任何依赖。

### 操作步骤
1. 选择年份类型（永历、公元、民国或伪清年号）
2. 输入年份数值
3. 点击"换算纪年"按钮
4. 查看换算结果，包括所有纪年系统的对应年份

## 技术实现

### 前端技术
- **HTML5**：页面结构
- **CSS3**：样式设计，包括响应式布局和美观的视觉效果
- **JavaScript (ES6)**：实现纪年换算逻辑和交互功能
